## Potsdamer Bürgerstiftung: Web Projects

This monorepo contains all public projects representing the Bürgerstiftung Potsdam on the web.

- **Website** - Main public website
- **Inselbühne** - The public website for our event location
- **Links** - A URL shortener service and QR-Code-Redirection tool to prevent QR-Codes from going "dead"
- **Portal** - A custom Directus instance and build modified to fit our needs
- **Shop** - Work in progress integration of MedusaJS
